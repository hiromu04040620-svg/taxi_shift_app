# ドメイン用語集

タクシー業界の専門用語の正典。実装・ドキュメント・UI文言の表記揺れを防ぐため、本ファイルを Single Source of Truth とする。

## 勤務形態

| 用語 | 読み | 英語識別子 | 説明 |
|---|---|---|---|
| 日勤 | にっきん | `dayShift` | 朝〜夕方の単発勤務。労働8h程度、月22〜24日 |
| 夜勤 | やきん | `nightShift` | 夕方〜深夜・早朝の単発勤務。労働8h程度、月22〜24日 |
| 隔日勤務 | かくじつきんむ | `alternateDay` | 日勤＋夜勤を連続。1回約20時間拘束、月11〜13乗務 |

**第一ターゲット: 隔日勤務**。日勤・夜勤も同じデータモデルで扱えるよう設計する。

## シフトサイクル関連

| 用語 | 説明 |
|---|---|
| サイクル | 勤務パターンの繰り返し単位。例: `[出, 明, 出, 明, 出, 明, 公]` の7日周期 |
| ライン / 班 | 同一サイクルで動くドライバーのグループ。A班、B班… |
| 起点日 | サイクルが開始される日付 |
| Override | サイクルから計算された予定を、特定日だけ上書きする例外データ |
| 出番変更 | 既存出番を別日に振り替えること。実体は2件のOverride |

## 給与・売上関連

| 用語 | 読み | 説明 |
|---|---|---|
| 営収 | えいしゅう | 1乗務または1ヶ月の総売上 |
| 税抜営収 | ぜいぬきえいしゅう | 消費税を除いた営収。歩合計算の基準になることが多い |
| 足切り | あしきり | 歩合給が発生する売上の下限額。これ未満は歩合ゼロ |
| 歩合 / 歩率 | ぶあい / ぶりつ | 売上に対する給与の比率 |
| 累進歩合 | るいしんぶあい | 売上が高いほど歩率が上がる賃金体系 |
| 実車率 | じっしゃりつ | 走行距離のうち客を乗せていた距離の割合 |
| 回数 | かいすう | 1乗務での乗車回数 |
| 水揚げ | みずあげ | 営収の俗称 |

## 法令関連（改善基準告示）

正式名称: **「タクシー・ハイヤー運転者の労働時間等の改善のための基準」**

| 項目 | 隔日勤務 | 日勤・夜勤 |
|---|---|---|
| 1日の拘束時間 | 2暦日21時間以内 | 原則13h、最大16h |
| 月の拘束時間 | 262h以内（労使協定で270hまで、年6ヶ月） | 299h以内 |
| 勤務後の休息 | 連続20h以上 | 連続8h以上 |
| 休日の連続時間 | 44h以上（20h＋24h） | 32h以上（8h＋24h） |
| 休日労働 | 2週に1回まで | 2週に1回まで |

本アプリでは上記を **目安として** アラート表示する。法的判断は会社の労務担当に確認するよう注記する。

## 期間区切り

| 用語 | 説明 |
|---|---|
| 月締め日 | 給与計算上の月の区切り。多くは15日締め（前月16日〜当月15日が1ヶ月） |
| 年シフト | 年間で先に組まれた固定シフト |
| 月シフト | 月単位で希望提出して組むシフト |
| 週シフト | 週単位で希望提出して組むシフト |
