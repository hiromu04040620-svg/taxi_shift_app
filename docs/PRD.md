# Product Requirements Document

## プロダクト名

（仮称）**TaxiShift** — タクシー乗務員のための個人用シフト＆売上管理アプリ

## 背景・課題

タクシー乗務員、特に隔日勤務者は以下の課題を抱える:

- 紙のシフト表を会社から渡されるが、スマホで一元管理する手段がない
- 売上を手書きの日報やメモアプリで記録しており、月次の振り返りや比較が困難
- 改善基準告示（月262h等）に対する自己管理の手段がない
- 出番変更や公出希望を出すとき、月の出番数や拘束時間を即座に把握しづらい

既存アプリを調査したが、**個人乗務員向けの統合管理アプリは見当たらない**（2026年6月時点）。
法人向けシフト管理SaaS（シフオプ等）は存在するが、運行管理者向けで個人ドライバーは使えない。

## ターゲットユーザー

- **プライマリ**: タクシー隔日勤務乗務員（28〜60歳、男性中心）
- **セカンダリ**: 日勤・夜勤のタクシー乗務員、ハイヤー運転手
- **想定スキル**: スマホは日常的に使うが、複雑なアプリは苦手な層も含む

## 価値提案

1. **サイクル登録で先々の予定が自動で見える** — 毎日入力不要
2. **出番ごとの売上を簡単記録 → 月次サマリーで可視化**
3. **改善基準告示の目安アラート** — 月の拘束時間オーバーを事前警告
4. **オフライン動作** — 営業中の電波の悪い場所でも記録可能

## 機能要件

### コア機能

#### 1. シフトサイクル管理
- 勤務形態選択（日勤 / 夜勤 / 隔日）
- サイクル定義（種別の配列＋起点日）
- プリセット提供（2出番2休7日周期、ダブ公含む9日周期、3勤3休＋ダブ公）
- 班ライン切替（複数サイクルの履歴管理、`validFrom`/`validUntil`）
- 月締め日カスタマイズ（デフォルト15日）

#### 2. カレンダー表示
- 月表示・週表示の切り替え
- シフト種別の色分け
- サイクルから自動計算した結果＋Overrideの合成表示
- 日付タップで詳細画面、長押しで種別変更

#### 3. シフト例外管理（Override）
- 任意の日付の種別を上書き
- 「元に戻す」でOverride削除
- 出番変更（2日分のOverrideをペアで管理）
- 公出希望・公休希望のフラグ管理（仮確定 / 確定）

#### 4. 売上記録
- 出番ごとに以下を記録:
  - 総営収、税抜営収
  - 内訳: 現金 / キャッシュレス / その他
  - 走行距離、実車距離
  - 乗車回数
  - 勤務開始・終了時刻、休憩時間
  - 燃料補給額（任意）
  - 自由記述メモ
- 写真添付（領収書・メーター画像、任意）

#### 5. 月次サマリー
- 出番数（実績／予定）
- 総営収、平均営収、最高営収日
- 実車率
- 月別推移グラフ（fl_chart）
- 歩合給の試算（足切り・歩率は設定値を使用）
- 改善基準告示の達成状況（拘束時間、連続休息）

#### 6. 改善基準告示アラート
- 月間拘束時間 90%/100% 超過警告
- 連続休息20h未満の検出
- 月の出番数上限超過警告
- 設定で個別ON/OFF可能

### 補助機能

#### 7. 設定
- 勤務形態
- サイクル定義・班変更履歴
- 月締め日
- 足切り額、歩率
- アラート閾値
- 表示名カスタマイズ（明け番→明休 等）
- テーマ（ライト/ダーク）

#### 8. 通知
- 出番前日リマインド
- 月末の拘束時間集計通知
- 月初の希望休出し忘れ防止通知（任意）

#### 9. データ管理
- バックアップ（端末内ファイル出力、JSON）
- 復元
- CSV エクスポート（月次売上）
- 機種変更時の引き継ぎ

#### 10. 収益化要素
- AdMob バナー広告（無料版）
- 「お得な情報」枠としてアフィリエイト連携（A8.net 経由）
  - タクシー転職、ドラレコ、ETCカード等
- プレミアム購入（買い切り or サブスク）で広告非表示

## 非機能要件

- **オフラインファースト**: ローカルDB（drift）で全機能動作。クラウド同期は将来検討
- **応答性**: カレンダー表示2秒以内、サマリー計算1秒以内
- **データ保全**: 起動時に自動バックアップ（直近3世代）
- **プライバシー**: 売上データは端末内のみに保持。外部送信なし
- **アクセシビリティ**: 動的フォントサイズ対応、コントラスト比 WCAG AA
- **対応OS**: iOS 15+ / Android 8+（API 26+）
- **言語**: 日本語のみ（v1）

## 非対応（v1スコープ外）

- 法人向け運行管理機能
- ドライバー間のシフト共有
- 配車アプリAPI連携（GO/Uber等）
- 自動売上取り込み（OCR等）
- マルチユーザー（家族・複数アカウント）
- クラウド同期
- 多言語対応

## 成功指標（KPI）

個人開発のためビジネス指標は控えめに設定:

- DAU/MAU比 25% 以上（継続利用の指標）
- 起動後1週間のリテンション 40% 以上
- ストア評価 4.0 以上

## リリース計画

単一バージョンとして全機能を実装。
リリース後の改善は GitHub Issues で管理。
